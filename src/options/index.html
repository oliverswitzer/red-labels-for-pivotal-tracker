What We Learned This Week for Pivotal Tracker

What does this extension do?

Once you've added your Pivotal Tracker API token below, this extension will:

  • Create a weekly "WWLTW for [This Friday's date]" chore at the top of your icebox
  • Prompt you to enter your learnings upon Finishing stories
  • Append your learnings to the description of the current week's WWLTW chore for easy copying (markdown and all!) into your email
How do I get started?

Enter your Pivotal Tracker API token below. You can find your API token right here on your Tracker user profile.
